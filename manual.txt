Инструкция по созданию файла для загрузки через TMRF-Firmware Update

1) Скомпилировать файл обновляемой прошивки (Microchip HEX)
2) Запустить программу "Шифратор прошивки TM-RF" (TMRF_FirmwareEncoder.exe)
3) В программе открыть файл HEX, и нажать кнопку "Шифровать". Появится окно сохранения шифрованного файла, 
   имя файла генерируется автоматически основываясь на версии хранящейся в HEX файле (в проекте файл firmware.h).
4) Сохранить шифрованную прошивку с расширением *.tmfw
5) Теперь прошивку (*.tmrf) можно загружать в устройство через бутлоадер



Процесс обновления

1) Установить TMRF-FirmwareUpdate (файл установщика - Setup.exe)
2) Запустить программу "Обновление прошивки TM-RF" (Пуск->Touchmemory->TMRF-FirmwareUpdate)
3) Подключить устройство к USB порту
4) Дождаться обнаружения устройства программой
5) Открыть шифрованный файл прошивки (*.tmrf) для загрузки в устройство
6) Нажать кнопку "Обновить"
7) Дождаться завершения процесса обновления.
